¿Qué es Markdown y por qué se utiliza en proyectos de software?

Markdown es un lenguaje de marcado ligero usado para dar formato a documentos de forma simple y rápida.
Permite crear textos estructurados sin herramientas complejas, usando solo texto plano y símbolos básicos.

Se utiliza en proyectos de software porque:

Permite documentar de forma clara.

Es compatible con GitHub, GitLab y sistemas de control de versiones.

Facilita escribir archivos como README, manuales y reportes.

Se puede previsualizar en casi cualquier plataforma.
