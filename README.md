# control-de-asistencias-.

Descripción del Proyecto

El Sistema de Registro de Asistencia es un proyecto académico que permite almacenar y consultar la hora de entrada y salida de los usuarios. Este proyecto demuestra el uso correcto de GitHub, control de versiones y documentación con Markdown.

Funciones del Sistema

Registrar hora de entrada

Registrar hora de salida

Consultar registros por fecha

Editar o eliminar datos

Visualizar historial de asistencia

Tecnologías Utilizadas

GitHub

Markdown

Organización del proyecto mediante ramas
